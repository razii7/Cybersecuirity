# 1. Clients and Servers

## What is a Client?

A client is a device or software application that requests services or resources from another device.

### Examples

* A web browser requesting a webpage
* An email application requesting messages
* A smartphone opening a website

Common client software:

* Web browsers
* Email clients
* Messaging applications

---

## What is a Server?

A server is a device or software application that provides services, resources, or information to clients.

### Examples

* Web server hosting websites
* Email server handling email delivery
* File server storing shared files

A server waits for requests and responds when clients ask for resources.

---

## Client and Server Roles

Communication often follows a client-server model.

Example:

1. User opens a web browser.
2. Browser sends a request.
3. Web server receives the request.
4. Server sends the webpage back to the browser.

Example:

Email Client → Email Server → Recipient

This model allows many clients to access centralized services.

---

# 2. Peer-to-Peer (P2P) Networks

## What is a Peer-to-Peer Network?

In a peer-to-peer network, devices communicate directly with each other without requiring a dedicated server.

Each device can act as both a client and a server.

### Example

Two computers sharing files directly on the same network.

---

## Advantages of P2P Networks

* Easy to set up
* Lower cost
* No dedicated server required
* Suitable for small networks

---

## Disadvantages of P2P Networks

* Less secure
* Difficult to manage as the network grows
* No centralized administration
* Backup and maintenance become challenging

---

# 3. Peer-to-Peer Applications

Peer-to-peer applications allow devices to exchange information directly.

### Examples

* File sharing applications
* Messaging platforms
* Collaboration tools

In these applications, devices may simultaneously send and receive data from other peers.

---

# 4. Multiple Roles in a Network

A single device can perform multiple roles.

### Example

A personal computer may:

* Act as a client when browsing websites.
* Act as a server when sharing files.
* Act as both simultaneously.

Modern devices often perform multiple networking functions depending on the services being used.

---

# 5. Network Components

A network consists of several major components working together.

The three primary categories are:

1. End Devices
2. Intermediate Devices
3. Network Media

---

## End Devices

End devices, also called hosts, are the source or destination of data communication.

They send, receive, or process data.

### Examples

* Desktop computers
* Laptops
* Smartphones
* Tablets
* Printers
* Servers

Each host uses an address that helps data reach the correct destination.

---

## Intermediate Devices

Intermediate devices connect end devices and help move data through the network.

### Examples

* Routers
* Switches
* Wireless access points
* Firewalls

These devices determine the best path for data and help manage network traffic.

---

## Network Media

Network media provides the path through which data travels.

### Examples

#### Wired Media

* Ethernet cables
* Fiber-optic cables

#### Wireless Media

* Wi-Fi
* Cellular networks

Media carries data between devices and network equipment.

---

# 6. Data Communication Example

The module demonstrated how an email travels through a network.

### Example Flow

1. A user sends an email from a computer.
2. The email leaves the sender's device.
3. It passes through local network devices.
4. Intermediate devices forward the data.
5. The data crosses other networks.
6. It reaches the recipient's local network.
7. The destination device receives the email.

This process shows how end devices, intermediate devices, and network media work together to deliver information.

---

# Key Takeaways

* Clients request services, while servers provide services.
* Peer-to-peer networks allow direct communication between devices.
* A device can perform multiple networking roles.
* Networks are built from end devices, intermediate devices, and media.
* Hosts use addresses to send and receive information.
* Data often travels through multiple devices and networks before reaching its destination.

# Additional Network Connectivity Options

## Internet Service Providers (ISPs)

### What is an ISP?

An Internet Service Provider (ISP) is an organization that provides access to the Internet and related services.

### Common ISP Services

* Internet access
* Email services
* Domain registration
* Web hosting
* Technical support
* Network connectivity

### Examples

* Home broadband providers
* Mobile network operators
* Enterprise internet providers

ISPs play a critical role in connecting users and organizations to the global Internet.

---

## ISP Connections

Users and organizations connect to the Internet through an ISP using various technologies.

The choice of connection depends on:

* Availability
* Cost
* Speed
* Reliability
* Geographic location

Once connected, the ISP acts as the gateway between the user's local network and the Internet.

---

## Cable and DSL Connections

### Cable Internet

Cable Internet uses the same infrastructure as cable television services.

#### Advantages

* High-speed connectivity
* Widely available in urban areas
* Suitable for streaming and online activities

#### Limitations

* Speeds may decrease during peak usage times due to shared bandwidth.

---

### DSL (Digital Subscriber Line)

DSL uses traditional telephone lines to provide Internet access.

#### Advantages

* Widely available
* Does not interfere with normal telephone usage
* Generally reliable

#### Limitations

* Performance decreases with distance from the service provider's equipment.
* Usually slower than modern cable or fiber connections.

---

## Additional Connectivity Options

### Cellular Networks

Cellular networks provide Internet access through mobile communication technologies.

Examples:

* 3G
* 4G
* 5G

#### Advantages

* Mobility
* Wide coverage
* Convenient access for smartphones and mobile devices

#### Limitations

* Data limits may apply
* Signal quality can affect performance

---

### Satellite Internet

Satellite Internet provides connectivity using satellites orbiting the Earth.

#### Advantages

* Available in remote locations
* Useful where cable, fiber, or DSL are unavailable

#### Limitations

* Higher latency
* Weather conditions may affect performance
* Often more expensive than terrestrial connections
